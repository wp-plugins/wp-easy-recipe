== Description ==

"WP EASY RECIPE" is a very simple plugin for add to "Recipe" content with category and tags on your site in a easy way.

Use [wp_recipe_list_page] sortcode for reterive all the recipe pages

== Installation ==

Step 1. Upload wp-easy-recipe folder to the `/wp-content/plugins/` directory

Step 2. Activate the plugin through the Plugins menu in WordPress

== Frequently Asked Questions ==

1.How publish all the recipe post on any spefic page?

Answer: Create a new page and add [wp_recipe_list_page] sortcode for get all the recipe pages

2.How update the social media content on details page?

Answer: Find the "WP Easy Recipe" settings from admin ("Settings" >> "WP Easy Recipe") and update the content of social media
